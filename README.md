# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

The Pizza Sales Analysis project is a SQL-based data analysis project that focuses on extracting meaningful business insights from a pizza restaurant's sales data. The project uses SQL queries to analyze customer orders, revenue, sales trends, pizza popularity, and order patterns.

The main objective is to understand customer purchasing behavior and help businesses make data-driven decisions to improve sales and profitability.

---

# 🎯 Objectives

- Analyze total sales and revenue.
- Find the total number of orders placed.
- Calculate average order value.
- Identify the best-selling pizzas.
- Determine the most and least ordered pizza types.
- Analyze sales by pizza category and size.
- Understand hourly and daily order trends.
- Calculate percentage contribution of each pizza category.
- Identify top revenue-generating pizzas.

# 📋 Sample SQL Queries

### Total Revenue

```sql
SELECT
ROUND(SUM(order_details.quantity * pizzas.price),2) AS Total_Revenue
FROM order_details
JOIN pizzas
ON pizzas.pizza_id = order_details.pizza_id;
```

---

### Top 5 Best Selling Pizzas

```sql
SELECT
pizza_types.name,
SUM(order_details.quantity) AS Total_Quantity
FROM pizza_types
JOIN pizzas
ON pizza_types.pizza_type_id = pizzas.pizza_type_id
JOIN order_details
ON pizzas.pizza_id = order_details.pizza_id
GROUP BY pizza_types.name
ORDER BY Total_Quantity DESC
LIMIT 5;
```
